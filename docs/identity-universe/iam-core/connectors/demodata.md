# Demo data

There are two dedicated demo data connectors for a non-existing Norwegian municipality named **Utkanten Kommune**. The connectors provide HR data and school information system data, and is intended for demonstration purposes. We do not recommend using these data for anything in production, but feel free to play around in your development tenants.

## Configuration inputs - Utkanten Kommune HR

| Input | Description | Example value |
|-|-|-|
| relativedate | A relative date for all start and end dates | 2026-08-01 |

## Configuration inputs - Utkanten Kommune SAS

| Input | Description | Example value |
|-|-|-|
| schoolyear | The starting school year of the connector | 2025/2026 |
