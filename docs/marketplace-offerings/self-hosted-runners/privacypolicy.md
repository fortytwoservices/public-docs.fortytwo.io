# Privacy Policy for Self-Hosted Runners (Windows and Ubuntu)

**Effective Date:** January 1, 2026

This Privacy Policy describes how Fortytwo ("we", "us", or "our") collects, uses, and shares information when you use our Self-Hosted Runners Windows and Ubuntu Virtual Machine images ("the Product") available on the Microsoft Azure Marketplace.

## 1. Information We Collect

When you deploy and use the Product within your Microsoft Azure environment, **we do not collect, access, or store any personal data, telemetry, or usage information** from your virtual machine. The Product operates entirely within your own Azure subscription and infrastructure.

## 2. Information Collected by Microsoft and Canonical

Because the Product is built on the Microsoft Windows or Ubuntu operating systems and deployed on Microsoft Azure, Microsoft and Canonical (for Ubuntu) may collect telemetry, diagnostic data, and usage information as part of the underlying OS and cloud platform operations. This data collection is governed by the [Microsoft Privacy Statement](https://privacy.microsoft.com/en-us/privacystatement) and the [Canonical Privacy Policy](https://ubuntu.com/legal/data-privacy). We do not have access to this data.

## 3. How Your Information is Used

Since we do not collect any data from the Product, we do not use your data for any purpose. You retain full control and ownership over all data, applications, and workloads you run on the Product.

## 4. Data Security

You are solely responsible for securing the virtual machine, managing access controls, and protecting the data stored within your Azure environment. We recommend following Microsoft's best practices for securing Azure Virtual Machines.

## 5. Changes to This Privacy Policy

We may update this Privacy Policy from time to time. Any changes will be reflected on this page with an updated "Effective Date."
